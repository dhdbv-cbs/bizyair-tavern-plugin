# BizyAir Tavern Plugin

SillyTavern 第三方扩展插件，用于在聊天正文中识别并生成图片，同时支持独立 LLM API 进行 Tag 生成与角色构建。

## 功能概览

- 基于 `image##...##` 标签触发 BizyAir 生图，并将结果回写到聊天正文/界面。
- 支持多模板切换（内置模板 + 自定义模板导入）。
- 支持多 API Key（英文逗号分隔）与 Key 池调度。
- 支持排队模式与请求节流（免费额度场景更稳定）。
- 独立 API Prompt Lab：
  - 剧情分析生成 `<角色>/<定位>`。
  - 按定位文本自动插入图片标签到正文。
  - 角色构建（基于角色卡 + 世界书）并回写角色预设。
- 预设系统（Jailbreak / Task / Character）支持：
  - 新建、编辑、另存为、复制、删除、导入、导出。
- 交互入口：
  - 双击 SillyTavern 左下角魔法棒（`Extensions` 按钮）打开独立工具弹窗。
  - 当独立 API 正在生成时，再次双击可中断当前请求。
- 移动端与桌面端布局适配（包含底部留白与弹窗可视优化）。

## 目录结构

bizyair-tavern-plugin/

## 安装方式

1. 将本目录放入：
   - `SillyTavern/public/scripts/extensions/third-party/bizyair-tavern-plugin`
2. 重启 SillyTavern 或刷新页面。
3. 在左侧菜单进入 `BizyAir` 面板进行配置。

## 基础配置

### 1) BizyAir 生图配置

- `API Key`：支持多个 Key，使用英文逗号分隔。
- `生图模板`：可选内置模板，也可导入自定义模板。
- `Web App ID`：默认可用，特殊模板可按需修改。

### 2) 独立 LLM API 配置

- `LLM URL`：OpenAI 兼容接口地址（示例：`https://api.openai.com/v1`）。
- `LLM API Key`：用于剧情分析与角色构建。
- `Model`：可拉取模型列表后保存。

## 使用流程

### A. 正文内标签生图

1. 在正文中插入类似：
   - `<image>image##tag1, tag2, tag3##<image>`
2. 插件扫描后会在对应位置注入“生成图片”按钮。
3. 点击生成，图片会显示在消息附近并写入画廊缓存。

### B. 独立 Tag 工具（魔法棒双击）

1. 双击左下角魔法棒按钮打开工具弹窗。
2. 可选：
   - `开始生图`：抓取最近上下文，调用独立 API 生成 `<角色>/<定位>`，并按定位插入正文标签。
   - `角色构建`：读取当前角色卡与世界书，生成角色资料预设。

## 开关说明

- `检测到 image## 时自动生成图片`：
  - 自动点击注入按钮触发生图。
- `正文生成后自动触发独立 API`：
  - 在角色消息渲染后自动执行独立 Tag 生成链路。
- `后台排队模式（最多同时发送 2 个请求）`：
  - 单 Key 免费用户建议开启，降低超队列报错概率。

## 预设导入与迁移

- 支持从旧 Bridge 备份 JSON 一键导入预设。
- 预设分类为：
  - `Jailbreak`
  - `Task`
  - `Character`
- 每个分类可单独导入/导出。

## 注意事项

- 多 Key 输入请使用英文逗号分隔，不建议混用中文标点。
- 独立 API 为空回复时，先检查：
  - URL / Key / 模型是否可用；
  - 任务预设与角色预设是否匹配当前场景。
- 若连续请求出现额度/排队问题，优先使用：
  - 多 Key 池；
  - 排队模式；
  - 降低自动触发频率。

## 已知限制

- 插件强依赖 SillyTavern 前端上下文与消息结构，版本差异可能影响行为。
- 不同 OpenAI 兼容网关对返回结构兼容性不完全一致，需按实际接口调优。
